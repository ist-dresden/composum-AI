Translation Process Debugging UI

With this form an automatic translation of a page or page tree can be done. It assumes the given path is a live copy of the original language (to be easily updateable from the blueprint) and will translate all properties heuristically determined to be text using ChatGPT, as well as change found paths to a copy with the appropriate language. Full documentation is available on the project page. Please be sure to understand this documentation.
