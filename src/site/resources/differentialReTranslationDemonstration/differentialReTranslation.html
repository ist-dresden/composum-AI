Differential Re-Translation POC

Proof of concept: explores how the reasoning capabilities of an LLM (ChatGPT) can support automatic adaption of new translations according to previous manual corrections. This is intended for the following scenario. A text is automatically translated, checked by the user and changed manually. Now a new version of the original comes in. The idea of the differential re-translation is to translate the new version of the original text automatically, while having the LLM keep the manual changes as far as possible.
